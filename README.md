# Config Schema Validator

[![Sponsor](https://img.shields.io/badge/Sponsor-GitHub%20Sponsors-ea4aaa?logo=githubsponsors)](https://github.com/sponsors/sdotwinter)

Validate JSON/YAML configuration files against schemas before deploy.

## Usage
```bash
python3 main.py validate --config samples/config.yaml --schema samples/schema.json
python3 main.py lint --config samples/config.yaml
python3 main.py report --config samples/config.yaml --schema samples/schema.json --json
```
